# ゴール設定インタビュー

3年/年間ゴールが「未確定」だらけのときに、対話で引き出すための問い。
**一度に全部埋めない。** 答えやすい問いから始め、固まった分だけ `../3年計画.md` / `../年間計画/<年>.md` に記入する。
抽象的な答えには「**具体的には?**」「**なぜ大事?**」「**どうなれば達成と分かる?**」を返して深掘りする。

## 1. ウォームアップ（広く）
- 3年後、何が変わっていたら「良い3年だった」と思える？
- 今いちばん時間を使っていることは？ 増やしたい？ 減らしたい？
- 直近で「これはうまくいった」と思えたことは？ なぜ？

## 2. 領域別（work / ai運用 / money / health）
- work: 3年後、どんな役割・働き方をしていたい？ 今と何が違う？
- ai運用: AI・自動化に何を任せ、自分は何に集中したい？
- money: 3年後の収入・資産・支出の「安心ライン」は？
- health: 3年後も続けたい習慣／やめたい習慣は？

## 3. 逆算（3年 → 今年）
- その3年後に近づくため、今年やると効くことを1〜2個だけ挙げると？
- 逆に、今年「やらないと決める」ことは？
- 今年の終わりに何があれば「前進した」と言える？（＝年間計画の的）

## 4. 記入
- 固まった領域から `../3年計画.md`（3年後の状態）と `../年間計画/<年>.md`（今年の的）に書く。
- 全領域そろわなくてよい。**1領域でも埋まれば朝夜ルーティンの逆算が回り始める**。
- 数値や期限が出せるものは入れる（後で「どうなれば達成か」を判定できる）。
