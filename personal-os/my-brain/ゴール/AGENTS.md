# ゴール（縦のladder）

3年→年間→デイリーの「的」と「履歴」を置く縦のladder。
横の領域別は `../areas/`、今の状態1枚は `../areas/現在地.md`（→ `ダッシュボード.md` に昇格予定・子02）。縦と横を混ぜない。

## 構成
- `3年計画.md`: 3年後にありたい状態（的）。
- `年間計画/<年>.md`: その年の的。3年計画から落とす。
- `デイリー/<年>/<月>/<年-月-日>.md`: 1日1枚・追記の履歴。
- `templates/デイリー.md`: デイリーの雛形。

## ルール
1. デイリーは1日1枚。`templates/デイリー.md` から作り、追記していく（上書きしない）。
2. デイリーには必ず「逆算チェック」を書く（今日やることが年間/3年目標に繋がるか）。
3. 進捗・AIジョブの現在状態はここに書かない。状態の1枚はダッシュボード（子02）。
4. 目標の中身（実際の3年/年間目標）は人間が記入する。AIは雛形と置き場まで。
5. secret / token / 認証値は書かない。
