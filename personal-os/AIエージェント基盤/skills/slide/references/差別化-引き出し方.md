# 差別化を引き出す問いかけパターン

「ありきたりなスライド」と「自分のものにしたスライド」を分けるための、対話テクニック集。

## 基本原則

1. **抽象的な答えが出たら一段具体に下ろす**
2. **教科書的な答えが出たら「あなた独自のは？」と返す**
3. **言語化が難しそうなら場面・相手を変える質問に切り替える**

## フレーム別の問いかけ

### 1. 独自視点を引き出す（Duarte POV）

```
✓ 「同じテーマで他の人が話したら、絶対に出てこない一文は？」
✓ 「業界の常識でA・Bと言われているとき、あなたはCと言える？」
✓ 「このテーマで、あなただけが知っている『裏側』は？」

❌ 「あなたの強みは？」（抽象的すぎる）
❌ 「差別化ポイントは？」（言語化が難しい）
```

### 2. 体験ストーリーを引き出す（Heath Concrete）

```
✓ 「このテーマで失敗したエピソードを1つ、いつ・どこで・誰と？」
✓ 「過去の自分に声をかけるとしたら何？」
✓ 「そのとき、何が一番悔しかった/嬉しかった？」

❌ 「実績を教えてください」（数字が出るだけ）
❌ 「成功体験は？」（盛りやすい）→ 失敗から聞く方が本質が出る
```

### 3. Empathy（聴衆の感情）を引き出す（Duarte Empathy）

3段階で深掘り:
```
段階1（属性）: 「聴衆の役職・業界・年齢・知識レベルは？」
   ↓
段階2（感情）: 「その人は今夜、何を考えながら眠りにつく？」
   「何に焦り・不安・希望を持っている？」
   ↓
段階3（抵抗）: 「このテーマに対して身構えている部分は？」
   「過去に似た提案で嫌な思いをした経験はありそう？」
```

### 4. Villain（敵）を引き出す（Gallo Villain）

日本人は対立を避けるので、言い換えで引き出す:
```
✓ 「このプレゼンの『批判すべきアンチパターン』は？」
✓ 「世の中で多くの人がやっているけど、間違いだと思うことは？」
✓ 「相手に絶対取ってほしくない選択肢は？」

❌ 「敵は？」（抵抗される）
❌ 「競合は？」（個人攻撃に流れがち）
```

### 5. STAR Moment を引き出す

Q1〜Q5で出た要素を素材に:
```
✓ 「終わった後、聴衆がSlackで誰かに送る『一瞬』は？」
✓ 「この資料で1枚だけ覚えてもらうとしたら、どのスライド？」
✓ 「自分でも忘れない一瞬を作れている？」
```

## 回答が「教科書的」だったときの返し方

```
ユーザー: 「DXは生産性を上げます」
↓
Claude: 「それは多くの人が言ってます。あなただけが言える独自の視点だと、もう一段先に何かありませんか？」
↓
ユーザー: 「うーん…意思決定の速度を上げる、とか」
↓
Claude: 「それです。それを30字以内で言うと？」
↓
ユーザー: 「DXは生産性を上げない。意思決定スピードを上げるだけだ」
↓
Claude: 「これがあなたのPOVですね。スライド全体の柱にします」
```

## 言語化が難しいときの場面転換

```
- 「もし1人の友人にこのテーマを語るなら？」
- 「酒の席で本音を言うとしたら？」
- 「過去の自分にアドバイスするとしたら？」
- 「10年後の自分が今の提案を見たらどう言う？」
```

場面を変えると本音が出やすい。

## 各フレームの組み合わせ推奨

| 用途 | 推奨フレーム組み合わせ |
|---|---|
| 営業提案 | 体験ストーリー + スタンス表明 |
| 講演登壇 | 独自視点 + STAR Moment（体験 or アンチテーゼ） |
| 自己紹介 | 体験ストーリー + 独自視点 |
| 経営報告 | 数値の独自解釈 + Big Idea圧縮 |
| 思想発信 | アンチテーゼ + 独自視点 |
