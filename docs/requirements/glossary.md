# Glossary

| Term | Meaning |
| --- | --- |
| Entry file | `CLAUDE.md` or `AGENTS.md`; a short router that is read by an AI coding tool. |
| Source of truth | The canonical file that owns a requirement, spec, progress state, contradiction, or decision. |
| Requirement ledger | The canonical table of requirement IDs, statuses, priorities, evidence, and notes. |
| Feature gate | A pre-implementation check for conflicts, scope growth, non-goals, and acceptance criteria. |
| Evidence | A code path, test result, screen check, commit, PR, or user decision that supports a status. |
| Non-goal | A deliberately out-of-scope item. |
| Contradiction | A conflict between requirements, specs, implementation, docs, or entry-file rules. |
