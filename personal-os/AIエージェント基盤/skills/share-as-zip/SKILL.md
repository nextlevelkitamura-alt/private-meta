# Share as ZIP

指定ファイルを、元ファイルを変えずに共有用ZIPへまとめる。

## 1. 役割

1. 対象は、この依頼で指定・添付されたローカルファイルだけにする。
2. 補足文があれば、ZIP内に `補足.txt` として加える。
3. デスクトップにはZIPだけを残し、個別ファイルのコピーは残さない。

## 2. 手順

1. 対象ファイルのローカルパスを確認する。読めない添付やパスが不明な時は、再添付またはパス指定を依頼する。
2. 各ファイル名を確認する。同名ファイルが複数ある時は、自動改名せずユーザーに確認する。
3. 先頭の元ファイル名を基準に、`<元ファイル名>-YYYY-MM-DD-HHmm.zip` を決める。ZIP名が既にある時だけ末尾に連番を付ける。
4. デスクトップ外の一時フォルダへ元ファイル名を保ったまま複製し、任意の `補足.txt` を加える。
5. 一時フォルダでZIPを作り、中身が元ファイル名と任意の `補足.txt` だけであることを確認してからデスクトップへ移す。
6. 一時フォルダを必ず消し、作成したZIPのパスと収録ファイル名を報告する。

## 3. 安全方針

1. 元ファイルは移動・編集・削除しない。
2. デスクトップに作るのはZIPだけとする。
3. 外部サービスへの送信・共有はしない。
4. 対象外のファイルやフォルダはZIPに含めない。
