# 業務スキル / global

Globalの業務スキル索引。正本は `skills/`、作成・移行・削除履歴は `logs/` を見る。

## スキル名: `project-slide-workflow`
概要: プロジェクト型の資料・スライドを、壁打ちで内容を固めながら1枚ずつ確認して作る。
近接・注意: 資料、LP、LINE、広告向けの成果物作成に使うため `applied`。

## スキル名: `video-transcription`
概要: 動画・音声から編集用timestamp JSONを作り、字幕ブロックやテロップ編集に使える形へ整える。
近接・注意: メディア制作成果物を直接作るため `applied`。

## スキル名: `html`
概要: 回答・作業結果を、人間が見る1枚もののHTML（レポート・調査まとめ・ダッシュボード・比較）に整え、self-contained な Artifact（PC前提）として出す。
近接・注意: 表示専用の成果物を直接作るため `applied`。正本(md)・同期対象(デイリー/計画)は対象外。メタ構造の説明・実装前合意は `meta-explain`（本Skillを出力規約として参照する側）。
