# source-template — 定型テンプレ（DM誘導等）

主に各アカウントの DM相談誘導ピラー（nextlevel⑤ / hiro_ai_dx④ 等）で使う。

## 用途

ハードCTA投稿。DMやリンクへ明確に誘導する。
連発するとフォロワー離れの原因になるため、**週1のみ** が原則。

## フロー

### Step 1: テンプレ選択

アカウントごとの基本テンプレを確認：

### nextlevel__career ⑤
```
「自分に合う仕事がわからない」
そんな方、DMで30分の無料キャリア相談やってます

▶ 派遣 / 正社員 / 紹介予定
あなたに合う形を一緒に考える

DMで「相談希望」とだけ送ってください
```

### hiro_ai_dx ④
```
毎週日曜に5社限定で
業務自動化の無料診断やっています

「何が自動化できるか分からない」
「何から始めればいいか分からない」
という状態で来てもらって大丈夫です

先週は飲食チェーンの発注管理と
EC会社のCS対応の話をしました

DMで「診断希望」とお送りください
（毎週日曜・無料・5社限定）
```

### Step 2: 直近のテーマや時期に合わせて軽くカスタム

完全テンプレ転用はNG。直近の状況で1〜2行更新：
- 季節（GW明け / 年度末 / 夏休み）
- 直近の事例（先週は何の話をしたか）
- 限定数（残席数等）

### Step 3: ハードCTAを明確に

- DMの**トリガーワード**を明示（「相談希望」「診断希望」等）
- 限定要素（人数 / 期間）があれば書く
- 何が得られるか具体（「30分の相談」「無料診断」）

### Step 4: 同月内の重複チェック

mode-publish.md の前に、同月内に DM誘導投稿が他にないか確認：
- 過去30日のスプシ G列で「⑤DM」or「④DM」をフィルタ
- 4件以上 → ユーザーに警告（投稿頻度を確認）

### Step 5: スプシ追記

mode-create-post.md の Step 5。G列=「⑤ DM相談誘導」or「④ DM誘導」。

## 注意点

- **連発NG**: ハードCTAは週1限定が原則
- **テンプレ完全転用NG**: 同じ文章の繰り返しはAI判定で表示が抑制されることがある
- **誇大表現NG**: 「絶対」「100%」「最強」など断定表現は避ける
- **限定要素の根拠**: 「5社限定」と書くなら本当に5社で締めること（信頼の問題）

## DM誘導の効果測定

- スプシ I〜L列で投稿のスコア確認
- 実際にDMが何件来たかをアカウントオーナーに聞く（DMはAIが自動取得不可）
- 効果が薄ければ次回テンプレを書き換え（accounts ファイルにメモ）
